# webgis

## 项目包安装
```
npm install
```

## 开发环境运行
```
npm run serve
```

## 打包运行
```
npm run build
```

## 文件

### src/components/onemap
```
onemap-area 行政区划
onemap-locate 查询定位
onemap-hegui 合规审查
onemap-paper 叠加图纸
onemap-topic-guihua 规划制作
onemap-topic-tugui 土规制作
onemap-topic-current 现状制作
onemap-topic-define 自定义图
onemap-unit 单元监测
```
